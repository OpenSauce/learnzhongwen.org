About Learn Zhongwen

Hi, I'm a software engineer who loves to learn new things. I'm currently learning Chinese and I'm using this app to help me learn. I hope you find it useful too.

This app is designed to provide resources and tools for learning Chinese, including reading, writing, listening, and speaking practice. Whether you're a beginner or an advanced learner, you'll find something useful here.

Features

  • Curated reading materials for different levels
  • Writing practice with stroke order guidance
  • Listening exercises with native speakers
  • Speaking practice with language exchange partners
